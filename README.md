# User Management System (Java Console)
Pequeña aplicación de consola desarrollada en Java que permite gestionar usuarios mediante un menú interactivo.

## Funcionalidades
- Crear usuarios con nombre y email
- Validación de email:
    - Formato básico correcto
    - Email único (no se permiten duplicados)
- Listar usuarios registrados de forma numerada
- Menú interactivo con control de errores

## Estructura del proyecto
- `Main` → Controla el flujo principal del programa
- `Sistema` → Lógica de negocio y gestión de usuarios
- `Usuario` → Modelo de datos
- `InputUtil` → Entrada de datos y validaciones

## Tecnologías utilizadas
- Java
- Programación orientada a objetos (POO)
- Listas (`ArrayList`)
- Git y GitHub

## Cómo ejecutar
1. Clona el repositorio
2. Ejecuta la clase `Main`
3. Interactúa con el menú desde la consola

## Estado del proyecto
Proyecto finalizado como práctica de fundamentos de Java y Programación orientada a objetos.
